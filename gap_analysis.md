## Overview

Gap analysis compares the **current state (AS-IS)** of global healthcare systems with the **desired future state (TO-BE)** as defined by WHO's Universal Health Coverage (UHC) targets and Sustainable Development Goal 3 (SDG-3).

## 5. Root Cause Analysis (5 Whys — Sub-Saharan Africa)

```
Why do people die from preventable diseases?
  └── Because they lack access to treatments
        └── Why? Because healthcare facilities are too far / unaffordable
              └── Why? Because infrastructure investment is inadequate
                    └── Why? Because government health budgets are underfunded
                          └── Why? Because GDP is low AND donor aid is inconsistent
                                = ROOT CAUSE: Poverty cycle + fragmented international aid
```
